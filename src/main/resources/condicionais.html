Condicionais na Programação em C

As condicionais são usadas para tomar decisões com base em certas condições. Na linguagem C, as principais estruturas condicionais são o if, else if, else e o switch.

1. Estrutura if

A estrutura if permite executar um bloco de código se uma condição for verdadeira.


#include <stdio.h>

int main() {
     int idade = 18;

     if (idade >= 18) {
          printf("Você é maior de idade.\n");
     }

     return 0;
}

Se a condição (idade >= 18) for verdadeira, a mensagem será exibida.

2. Estrutura else

A estrutura else é usada junto com o if para executar um bloco de código se a condição do if for falsa.


#include <stdio.h>
int main() {
     int idade = 16;

     if (idade >= 18) {
          printf("Você é maior de idade.\n");
     } else {
          printf("Você é menor de idade.\n");
     }

     return 0;
}

Se a condição (idade >= 18) for falsa, a mensagem "Você é menor de idade." será exibida.

3. Estrutura else if

A estrutura else if é usada para testar múltiplas condições.

#include <stdio.h>

int main() {
     int nota = 75;

     if (nota >= 90) {
          printf("Nota A.\n");
     } else if (nota >= 80) {
          printf("Nota B.\n");
     } else if (nota >= 70) {
          printf("Nota C.\n");
     } else {
          printf("Nota D.\n");
     }

     return 0;
}

O programa verifica as condições na ordem e executa o bloco de código do primeiro if ou else if cuja condição for verdadeira.

4. Estrutura switch

A estrutura switch é uma alternativa ao else if para testar uma variável contra diferentes valores.

#include <stdio.h>

int main() {
     int dia = 3;

     switch (dia) {
          case 1:
          printf("Domingo\n");
          break;
     case 2:
          printf("Segunda-feira\n");
          break;
     case 3:
          printf("Terça-feira\n");
          break;
     default:
          printf("Outro dia\n");
     }

     return 0;
}

O switch compara a variável dia com cada case. Quando encontra um case correspondente, executa o bloco de código associado. O break é usado para sair da estrutura switch após executar o bloco de código do case correspondente.
