Comandos de Repetição na Programação em C

Os comandos de repetição são usados para executar um bloco de código várias vezes. Na linguagem C, os principais comandos de repetição são o for, while e do...while.

1. Estrutura for

A estrutura for é usada quando sabemos com antecedência quantas vezes queremos repetir um bloco de código.


#include <stdio.h>

int main() {
    for (int i = 0; i < 10; i++) {
         printf("Número %d\n", i);
    }
    return 0;
}

O loop for inicializa a variável i em 0, verifica a condição i < 10 e incrementa i em 1 a cada iteração, executando o bloco de código até que a condição seja falsa.

2. Estrutura while

A estrutura while repete um bloco de código enquanto uma condição for verdadeira.


#include <stdio.h>

int main() {
    int i = 0;
    while (i < 10) {
         printf("Número %d\n", i);
         i++;
    }
    return 0;
}

O loop while verifica a condição i < 10 antes de cada iteração e executa o bloco de código enquanto a condição for verdadeira.

3. Estrutura do...while

A estrutura do...while é semelhante ao while, mas verifica a condição após cada iteração, garantindo que o bloco de código seja executado pelo menos uma vez.


#include <stdio.h>

int main() {
    int i = 0;
    do {
         printf("Número %d\n", i);
         i++;
    } while (i < 10);
    return 0;
}

O loop do...while executa o bloco de código uma vez antes de verificar a condição i < 10 após cada iteração.
