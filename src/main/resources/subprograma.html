Subprogramas na Programação em C

Subprogramas, também conhecidos como funções ou procedimentos, são blocos de código que realizam uma tarefa específica e podem ser chamados a partir de outras partes do programa. Em C, os subprogramas ajudam a organizar e reutilizar o código.

1. Definição de Função

Uma função é definida com um tipo de retorno, um nome e parâmetros opcionais. Aqui está um exemplo de função que soma dois números:


#include <stdio.h>

int soma(int a, int b) {
    return a + b;
}

int main() {
    int resultado = soma(5, 3);
    printf("Resultado: %d\n", resultado);
    return 0;
}

A função soma recebe dois inteiros como parâmetros e retorna a soma deles. No main, a função é chamada com os valores 5 e 3.

2. Funções sem Retorno (void)

Funções que não retornam um valor têm o tipo de retorno void. Elas podem executar ações, como exibir uma mensagem:


#include <stdio.h>

void saudacao() {
    printf("Olá, mundo!\n");
}

int main() {
    saudacao();
    return 0;
}

A função saudacao não retorna nenhum valor e simplesmente imprime "Olá, mundo!".

3. Passagem de Parâmetros

Os parâmetros podem ser passados por valor ou por referência. Aqui está um exemplo de passagem por referência usando ponteiros:


#include <stdio.h>

void incrementar(int *n) {
    (*n)++;
}

int main() {
    int numero = 5;
    incrementar(&numero);
    printf("Número incrementado: %d\n", numero);
    return 0;
}

A função incrementar recebe um ponteiro para um inteiro e incrementa o valor apontado. No main, o endereço de numero é passado para a função.

4. Funções Recursivas

Funções recursivas chamam a si mesmas. Um exemplo clássico é a função para calcular o fatorial de um número:


#include <stdio.h>

int fatorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * fatorial(n - 1);
    }
}

int main() {
    int numero = 5;
    printf("Fatorial de %d: %d\n", numero, fatorial(numero));
    return 0;
}

A função fatorial chama a si mesma até que n seja 0, retornando 1. Para qualquer outro valor, retorna n multiplicado pelo fatorial de n-1.
